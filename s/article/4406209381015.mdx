title: "Domo University (Docebo) Admin FAQ"
---

The following are some common customer problems and solutions in Domo University:

1. **Training module content not loading completely.**
   1. Chrome changed the default settings for cookies and this seems to be affecting page loading for some users.
   2. Ensure that the “Block third-party cookies” setting is disabled at chrome://settings/content/cookies.
   3. Also try:
      1. Clear cache/cookies.
      2. Disable pop-up/ad blockers.
      3. Set page exception for VPN.
      4. Use a different browser.
2. **Course Player is cropping slides in a training course.**
   1. Have users change the zoom % in their browser.
   2. Resize browser window.
   3. Reload page.
   4. Try another browser.
3. **[Coupon code](https://help.docebo.com/hc/en-us/articles/360020080080-Managing-Coupons-for-E-Commerce) not working**.
   1. Check expiration dates.
   2. Check assigned courses.
4. **User stuck on [exam waitlist](https://help.docebo.com/hc/en-us/articles/360020084340-Managing-Self-Enrollments-and-Waiting-Lists).**
   1. Select Course Management from the admin menu in Docebo.
   2. Click on the Waitlist icon in the row next to the relevant exam.
   3. Select the user and Change Status to Subscribed from the Choose Action drop-down.
5. **Student exam access**.
   1. Students will gain access to the Data Specialist Level 1 exam upon completion of the [Student Training course](https://learndomo.domo.com/learn/course/internal/view/elearning/403/student-training) in the Learn by Role Catalog.
   2. On occasion, a module may not register as complete after it has been completed. Administrators can [mark modules complete](https://help.docebo.com/hc/en-us/articles/360020080200-Checking-the-Course-Report#subtitle-3) in Course Management by selecting Reports from the course drop-down (three dots) and then searching for the user. Click on the users' name and then change the module status to Complete.
6. **[Enrolling Users](https://help.docebo.com/hc/en-us/articles/360020124659-Enrolling-Users-into-Courses) into Courses.**
7. **[Edit Enrollment](https://help.docebo.com/hc/en-us/articles/360020124659-Enrolling-Users-into-Courses#subtitle-5) Deadlines**
   1. Got to Course Management and find the relevant course.
   2. Select Enrollments and find the user. Select edit (three dots) and Edit Enrollment.
   3. Click the Enable Deadline box and set new date range. Click Confirm before exiting the page.
8. **Problems [submitting files](https://help.docebo.com/hc/en-us/articles/360020084440-Creating-Tests-and-Managing-Test-Questions#subtitle-8) in the practical portion of an exam**.
   1. The system limits users to nine files and the exam may require more. As per the instructions, combine the files into a single ZIP before submission to avoid this problem.
   2. If users submit incorrectly, then manually fail the practical portion of their exam, and have them resubmit a single zip file. This will count as their second attempt. If they don’t pass, then the [exam can be manually reset](https://help.docebo.com/hc/en-us/articles/360020126719-Resetting-the-Completion-of-a-Training-Material-for-Learners-in-a-Course) to provide their actual second attempt.
9. **Data Specialist exam - Workbench supplement for Mac users.**
   1. Supplemental content to support certification for Mac Users that DO NOT Have Access to Workbench.
   2. Users will need to be added to the Workbench catalog and then enrolled in the [Workbench Supplement](https://learndomo.domo.com/learn/course/393/Workbench%2520Supplement%2520%28Mac%29%2520-%2520Domo%2520Certification%2520v2.0%2520) course. Users will be given a DataSet after the test.
10. **Exam feedback from users.**
    1. Please forward exam feedback from users to Nick Alfonso so we can add it to our exam development plan.
11. **[Merging user accounts](https://help.docebo.com/hc/en-us/articles/360020082480-How-to-Merge-User-Profiles) in Docebo to consolidate user activity.**
