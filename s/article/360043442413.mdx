## Intro

DataSet Merge is another update method that can be added only to certain connector types and allows you to perform an upsert on that DataSet. This feature is turned on at the DataSet level and not instance wide.

## How to request the feature

After a support ticket has been submitted to support to turn this feature on, you will need to submit the request through JIRA with the following information:

- Customer use case
- The name of and a link to the DataSet
- The unique key in the DataSet for the Merge function to use
- A rough estimate of how often the DataSet will run
- A rough estimate of how much data will the DataSet process each run

After the JIRA is completed and submitted, assign it to Greg Pirie for review and completion.
